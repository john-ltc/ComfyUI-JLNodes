# ComfyUI-JLNodes

Quality-of-life custom nodes for ComfyUI.

Small utility nodes designed to simplify common workflows.

## Nodes

- **s3_image_node**  
  Uploads generated images directly to Amazon S3.

- **s3_video_node**  
  Uploads generated videos to Amazon S3.

- **azure_image_node**  
  Uploads generated images to Azure Blob Storage.

- **azure_video_node**  
  Uploads generated videos to Azure Blob Storage.

- **latent_save_output_node**  
  Saves latents to disk while also passing them through as output.

- **save_conditioning_json**  
  Saves prompt conditioning data to a JSON file.

- **load_conditioning_json**  
  Loads conditioning data from a JSON file.
